# 스왑 UI/UX 스펙

> FR-7a 세부 UX 규칙. 오프닝 절차는 `opening.md` 참고.

---

## 1. 스왑 시점 UI 동작

- 스왑 권리자 차례에 **"Swap" 버튼** 하나만 표시한다.
- "패스" 버튼은 표시하지 않는다.
- 보드에 돌을 두면 **스왑 없이(패스)** 착수가 진행된다.
- "Swap" 버튼을 누르면 색을 교환하고 다음 단계로 진행한다.

## 2. 안내 문구

스왑 가능 상태에서 다음 문구를 표시한다:

> "색을 교환할 수 있습니다. 그냥 돌을 두면 교환 없이 진행됩니다."

## 3. 근거

- "패스" 버튼은 별도 액션이 아님 — 돌을 두는 행위 자체가 패스.
- 버튼이 하나뿐이므로 UI가 단순해지고 혼란이 줄어든다.
