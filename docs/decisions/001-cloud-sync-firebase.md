# 001 — 기보·프로필 클라우드 동기화: Firebase Spark 플랜 채택

- **상태**: 결정됨 (미착수)
- **결정일**: 2026-06-14

## 배경

현재 기보·프로필 데이터는 `localStorage`에 저장된다. 기기/브라우저가 달라지면 데이터가 공유되지 않는다.

## 결정

클라우드 동기화 도입 시 **Firebase Spark 플랜(무료)** 을 채택한다.

- **Firebase Auth**: 구글 소셜 로그인. 신용카드 불필요.
- **Firestore**: 기기 간 기보·프로필 동기화.
- **무료 한도**: 읽기 5만/일, 쓰기 2만/일, 저장 1GB — 현재 규모에서 초과 없음.

## 구현 방향 (착수 시 참고)

- 로그인 전: 기존 `localStorage` 폴백 유지.
- 로그인 후: Firestore에 저장·조회, 기기 간 공유.
- GitHub Pages 정적 배포 유지 (Firebase Hosting 불필요).
